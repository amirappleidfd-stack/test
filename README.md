# 🚀 نصب و اجرای Marzban روی Railway

این پروژه یک نسخه فورک‌شده از **Marzban** است که برای اجرای آسان روی پلتفرم **Railway** آماده شده است.

با این روش می‌توانید Marzban را روی Railway دیپلوی کرده و پنل مدیریت Xray را بدون نیاز به سرور جدا اجرا کنید.

---

## ✨ امکانات

* اجرای Marzban روی Railway
* اجرای خودکار Xray Core
* دیتابیس داخلی SQLite
* ساخت ادمین از طریق Console
* مناسب برای مدیریت کاربران و کانفیگ‌های VPN

---

# 🛠️ روش نصب

## 1. Fork کردن پروژه

ابتدا این ریپازیتوری را در GitHub خودتان Fork کنید.

بعد از Fork، یک نسخه مستقل از پروژه داخل GitHub شما ساخته می‌شود.

---

## 2. Deploy روی Railway

1. وارد Railway شوید.
2. گزینه **New Project** را انتخاب کنید.
3. گزینه **Deploy from GitHub Repo** را بزنید.
4. ریپازیتوری Fork شده خودتان را انتخاب کنید.
5. منتظر بمانید تا Build و Deploy کامل شود.

---

# 👤 ساخت یوزر ادمین

بعد از Deploy، وارد Console پروژه در Railway شوید و دستور زیر را اجرا کنید:

```bash
marzban-cli admin create --sudo
```

بعد از اجرا:

* نام کاربری ادمین را وارد کنید.
* رمز عبور را تعیین کنید.

حالا اکانت مدیریت ساخته شده و می‌توانید وارد پنل شوید.

---

# 🌐 ورود به پنل مدیریت

بعد از Deploy موفق، Railway یک دامنه اختصاصی برای پروژه ایجاد می‌کند.

برای ورود به پنل، در انتهای دامنه عبارت زیر را اضافه کنید:

```
/dashboard
```

مثال:

```
https://your-project.up.railway.app/dashboard
```

سپس با نام کاربری و رمز عبوری که ساخته‌اید وارد پنل Marzban شوید.

---

# ⚙️ تنظیمات

برای تغییر تنظیمات می‌توانید از بخش:

```
Railway → Variables
```

متغیرهای محیطی مورد نیاز را اضافه کنید.

بعد از تغییرات، پروژه را دوباره Deploy کنید.
